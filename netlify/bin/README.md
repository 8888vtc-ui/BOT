# GNUBG Binary Placeholder

Place the GNUBG Linux binary here.

## How to get it:

1. Download from: https://www.gnu.org/software/gnubg/
2. Or compile from source
3. Name it `gnubg` and place it here
4. Make it executable: `chmod +x gnubg`

The binary must be:
- Linux x64
- Statically linked (no dependencies)
- Executable in Lambda environment
